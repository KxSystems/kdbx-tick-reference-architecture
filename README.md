## Template .env
```
export SCHEMA_DIR=/path/to/data/directory/schemas
export TPLOG_DIR=/path/to/data/directory/tplogs
export HDB_DIR=/path/to/data/directory/hdb
export PROCESS_LOG_DIR=/path/to/data/directory/proclogs
export TICK_PORT=5010
export RDB_PORT=5011
export HDB_PORT=5012
export GW_PORT=5013
```

Requires `PROCESS_LOG_DIR` directory to be created.
